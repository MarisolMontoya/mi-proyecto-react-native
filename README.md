## Reflexión: Ventajas de los Custom Hooks

Los Custom Hooks permiten separar la lógica de negocio de la parte visual de una aplicación. 
En este proyecto, el hook useProductos se encarga de manejar la carga de productos, controlar el estado de carga y obtener la información del artesano relacionado.

Esto permite que los componentes sean más organizados, fáciles de mantener y que la lógica pueda reutilizarse en diferentes pantallas sin repetir código.
